# 📑 SQL Daily Drills

Repositorio dedicado a la práctica diaria de SQL Server, enfocado en la resolución de problemas lógicos, optimización de consultas y modelado de datos.

### 🛠️ Entorno de Trabajo
* **Base de Datos:** Northwind / AdventureWorks
* **Motor:** SQL Server 2019
* **IDE:** SQL Server Management Studio (SSMS) / Datagrip

### 📂 Estructura de Práctica
* `/scripts`: Consultas de ejercicios diarios.
* `/procedimientos`: Lógica programable (Stored Procedures, Functions).
* `/retos`: Casos de negocio complejos.

### ⏱️ Rutina
1. Revisar teoría.
2. Resolver ejercicios prácticos en sesión local.
3. Documentar y pushear cambios.
